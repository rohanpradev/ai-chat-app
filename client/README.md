# Client

React client for the chat app.

Use the root README for setup and deployment. Most commands are intended to run from the repository root.

## Development

```bash
bun install --frozen-lockfile
bun run --filter @chat-app/client dev
```

The Vite dev server runs on `http://localhost:5173`.

## Build and Checks

```bash
bun run --filter @chat-app/client build
bun run --filter @chat-app/client typecheck
bun run --filter @chat-app/client test
```

## Notes

- Routes are generated by TanStack Router.
- Shared request/response types come from `@chat-app/shared`.
- Mermaid support is loaded lazily when markdown contains Mermaid fences.
- Production serving is handled by the repo root `Dockerfile` and Nginx config in this workspace.
